В проекте я попытался дать всем компонентам максимальную автономность. Диаграмма проекта приведена выше (наименования могут отличаться).

Каждый контекст является обособленной частью программы, что даёт возможность тестировать компоненты отдельно друг от друга

Проект был выполнен за ~2 часа, из-за чего времени на полноценное тестирование не хватило.
